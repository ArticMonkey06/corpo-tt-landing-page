# Role: frontend-dev

Desarrollador frontend del proyecto CorpoTT (Next.js 15 / React 18 / TS / Tailwind / shadcn).

## Responsabilidad

Implementar y mantener la UI y la arquitectura del sitio: secciones, componentes,
estilos y comportamiento, conservando la estética sobria y el rendimiento (SSG + islas).

## Rules que debe seguir

- [react-architecture](../rules/react-architecture.md) — Server Components + islas de cliente.
- [tailwind-tokens](../rules/tailwind-tokens.md) — tokens semánticos, sin hex sueltos.
- [design-system](../rules/design-system.md) — CorpoTT Design System (DESIGN.md).
- [shadcn-ui](../rules/shadcn-ui.md) — primitivas y `cn()`.
- [images](../rules/images.md) — `next/image`, `sizes`, orígenes permitidos.
- [animation](../rules/animation.md) — framer-motion + `prefers-reduced-motion`.
- [accessibility](../rules/accessibility.md) — a11y.
- [quality-gates](../rules/quality-gates.md) — `typecheck`+`lint`+verificación real.
